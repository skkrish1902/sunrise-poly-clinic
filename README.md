# Sunrise Poly Clinic

This is the Sunrise Poly Clinic project, an Angular application designed to manage and provide information for a polyclinic.

## Project Structure

The project follows a standard Angular structure with the following key directories and files:

- `src/app`: Contains the main application components and modules.
- `src/assets`: Directory for static assets like images and fonts.
- `src/environments`: Contains environment-specific configurations.
- `src/index.html`: The main HTML file for the application.
- `src/main.ts`: The entry point for the Angular application.
- `angular.json`: Configuration file for Angular CLI.
- `package.json`: Lists project dependencies and scripts.
- `tsconfig.json`: TypeScript configuration for the project.

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**
   ```
   cd sunrise-poly-clinic
   ```

3. **Install dependencies:**
   ```
   npm install
   ```

4. **Run the application:**
   ```
   ng serve
   ```

5. **Open your browser and navigate to:**
   ```
   http://localhost:4200
   ```

## Features

- User-friendly interface for managing clinic information.
- Responsive design for accessibility on various devices.
- Environment-specific configurations for development and production.
